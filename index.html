Discover Endless.JS Plugin!

Hello webmaster! Are you here because you search an original plugin for your website? This article is a demonstration of my jQuery plugin "Endless.JS". Endless.JS allows to do websites designed in cycle.

The idea is to join the top with the bottom. I hope in the next version you'll be able to join the left with the right. Your website will allowed to look like a sphere^^.

The installation is very easy. You must just integrate endless.js in the HTML head (after to have import jQuery Library) of your website and to create a little JavaScript snippet to specify which elements will be endless. You can view the source-code of this page by right click in doubt case.

Example :

<!-- Import jQuery Library -->

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script>

<!-- Integrate endless.js plugin -->

<script type="text/javascript" src="js/endless.js"></script>

<!-- Script using endless.js -->

<script type="text/javascript">

$(document).ready(function(){

$('yourElement').endless({

direction:'duo', //Direction : up (infinite top scrolling), down (infinite bottom scrolling), duo (infinite top and bottom scrolling)

scrollbar:'disable' //Enable or disable the scrollbar

});});</script>

Plugin is available on GitHub

Endless.js is actually in development. It's advisable to not use id in your HTML code element which is in endless, prefer classes.
